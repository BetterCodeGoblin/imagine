# Unreal Reboot Plan

## Goal
Rebuild IMAGINE as an Unreal Engine 5 project with a cleaner gameplay core and less engine-fragile scaffolding.

## Phase 1: Foundation
- Create UE5 project
- Establish naming conventions
- Set up source control expectations for Unreal
- Define first playable slice

## Phase 2: First Vertical Slice
- Player controller / movement
- Health or endurance tracking
- One effort-driven progression mechanic
- One environment or traversal challenge
- One feedback layer (UI, VFX, or narration)

## Phase 3: Exercise-Aware Expansion
- Reintroduce machine-informed inputs only when they improve a working loop
- Evaluate hardware bridge architecture for Unreal
- Keep integrations modular and optional

## Design Principle
Do not recreate the entire previous Unity scaffold. Rebuild only what the playable slice needs.
